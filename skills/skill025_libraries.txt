## [1] APPROVED LIBRARIES
- **uv**: Ultrafast dependency installer (though pip is default for Python 3.14.3).
- **Ruff**: All-in-one linter and formatter. Replaces flake8, isort, black.
- **Pytest + Playwright**: Native testing and End-to-End (E2E) UI testing.
- **Pulumi**: Python-driven infrastructure as code (AWS, GCP, Azure).
- **GitHub Actions**: Automated CI/CD execution pipeline.
