## [0] METADATA (ADR)
- Problem: Visual fragmentation and inconsistent brand representation.
- Decision: Mandate Mistica Design System with Confianza23 visual identity.
- Mechanism: Strict mapping of assets, colors, and icon paths.
- Consequence: Enterprise-grade UI consistency and brand alignment.

---

- **Primary Logo**: https://www.confianza23.es/assets/images/logotipo.png
- **Placement**: Header (Top-Left) with proportional scaling.
- **Visual Assets Directory**: `assets/images/` 

## [2] DESIGN SYSTEM (MÍSTICA)
- **Framework**: Mistica (Telefónica Design System).
- **Core Palette**:
  - **Mistica Blue (Primary)**: #001935 (Headers, Surfaces, Filled Icons).
  - **Mistica Accent (Action)**: #00a9e0 (Regular Icons, Links, CTA).
  - **Background (Surface)**: #f0f4f8 (Light Blue-Grey).
- **Component Strategy**: Server-side rendering (HTMX) or Next.js React Server Components.

## [4] IMPLEMENTATION CONSTRAINTS
- **Icon Styling**:
  - `Regular`: Use `stroke: #00a9e0` (stroke-width: 1.5).
  - `Filled`: Use `fill: #001935`.
- **Transitions**: Smooth (cubic-bezier) 0.3s.
- **Interactions**: On-Hover `translateY(-5px)`
- **Security**: Mandatory sanitization of HTML if using `unsafe_allow_html=True` for SVGs
